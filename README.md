## Setting up

```
From the terminal, enter the folder where you want to keep the project and perform the following steps:

$ git clone https://github.com/souzasmatheus/twistedideas.git
$ cd twistedideas
$ cp .env.sample .env
$ yarn
```

## Running

```
$ cd twistedideas
$ yarn start
```
### Running Tests

```
$ cd twistedideas
$ yarn test
```
